Zend_Console
===================
* Zend Framework Cli
* Zend Framework multi database

Requirements
--------
* PHP 5.3+
* Zend Framework 1.1+

Configure
----------
* public/index.php  `#!/usr/local/zend/bin/php`
* php.ini `include_path = "/usr/local/zend/share/ZendFramework/library"`

How to use
-----------
* ./index.php --help
* ./index.php -a ${controller}.${action}
